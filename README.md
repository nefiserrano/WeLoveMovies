Link to the live website: https://nefiserrano.github.io/WeLoveMovies/

We Love Movies will be a web application that allows users to track movies they want to watch and keep a record of movies they’ve already seen. 
Users can add movies manually, fetch movie information from a public API, and organize their watchlist by genre, rating, or status. 
The app focuses on a smooth, responsive, and visually appealing experience, making it easy for users to browse, add, and view details of movies.
